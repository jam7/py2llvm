====================================
py2llvm: A Python to LLVM translator
====================================

Author: Syoyo Fujita


Requirements
============

  - python 2.4 or above
  - LLVM 2.3
  - llvm-py 0.2.1

Status
======

  - very early stage. Almost all features doesn't work ;-)


How to use
==========

  $ ./py2llvm <input.py>
